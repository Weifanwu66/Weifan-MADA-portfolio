After looking at the variables, the interesting questions I could think of include 1.Is there any pattern of age difference throughout the years (e.g. increasing age difference as time goes by)? 2. Are males always older than females as Hollywood couples? 3. Is there any same-gender couple with age difference? 4. Is age difference associated with directors (preference). 5. Which actors (actresses) always play characters who are involved in romantic relationships with age gap? Since the column names are clean, the data structure seems correct, and there is no missing value, it seems like the author who cleaned this data did a good job. We can start to explore some data For now, let’s try to answer the first by selecting the related variables exploratory data analysis report

Let’s see if there’s any director who has directed multiple movies with big-age-difference romantic relationships (over 20 years).

director_pref=age_gaps%>%
  select(director,age_difference)%>%

  count(director)%>%

director_chara=semi_join(age_gaps,director_pref,by="director")
